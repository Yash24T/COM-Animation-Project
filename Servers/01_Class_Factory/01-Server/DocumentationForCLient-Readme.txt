DocumentationForCLient-Readme.txt

1. Providing .h, .dll, .reg files along with this documentation.
2. Copy the dll to your C:\Windows\System32 directory.
3. In the .h (header) file you will find the necessary interfaces and functionality.
	a. SumOfTwoIntegers() takes two integers value as firt two parameter values and gives the result as sum in the third parameter as out parameter.
	b. SubtractionOfTwoIntegers() takes two integers value as firt two parameter values and gives the result as sum in the third parameter as out parameter.
4. Use the interfaces and their functionality in your code as per COM rules.
5. After doing all the above things Double click on the .reg file to register the DLL before running your client program.
